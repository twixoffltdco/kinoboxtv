Часто задаваемые вопросы

❓ Как найти ID фильма на Кинопоиске?
Откройте страницу фильма на kinopoisk.ru — в адресной строке будет ссылка вида https://www.kinopoisk.ru/film/12345/. Число 12345 — это и есть нужный ID.
❓ Как открыть фильм через KinoBox?
Введите в адресную строку браузера: kinobox.tv/films/ID для фильма или kinobox.tv/serials/ID для сериала. Плеер загрузится сам.
❓ Почему плеер не загружается?
Убедитесь, что ID в адресной строке — это корректное число. Иногда источник может быть временно недоступен — попробуйте переключить источник в меню плеера (кнопка выбора выше видео).
❓ Нужна ли регистрация?
Нет. KinoBox полностью бесплатен, регистрация не требуется.
❓ Работает ли на телефоне?
Да, интерфейс полностью адаптивен. Плеер корректно работает на iOS и Android браузерах.
❓ Можно ли смотреть сериал по сериям?
Да. При открытии сериала в плеере появляется меню выбора сезона и серии.
❓ Что делать, если видео не воспроизводится?
Попробуйте: 1) Сменить источник в меню плеера. 2) Отключить блокировщик рекламы. 3) Очистить кэш браузера. 4) Попробовать другой браузер.
❓ Есть ли субтитры?
Наличие субтитров зависит от конкретного источника. Некоторые источники предоставляют субтитры на нескольких языках — проверьте настройки внутри плеера.
❓ Как встроить плеер на свой сайт?
Смотрите раздел 📘 Docs — там описана интеграция через скрипт kinobox.js.

Документация KinoBox

📌 Маршруты URL

KinoBox использует простые маршруты на основе ID Кинопоиска:

/films/{kinopoisk_id} — воспроизведение фильма
/serials/{kinopoisk_id} — воспроизведение сериала

Пример: kinobox.tv/films/301234

⚙️ Быстрая интеграция

Вставьте следующий код на любую страницу, чтобы добавить плеер:

<!-- Подключите стили -->
<link rel="stylesheet" href="https://kinobox.tv/kinobox.css">

<!-- Контейнер плеера -->
<div id="myPlayer"></div>

<!-- jQuery (если ещё не подключён) -->
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

<!-- Скрипт KinoBox -->
<script src="https://kinobox.tv/kinobox.js"></script>

<script>
  kinobox('#myPlayer', {
    search: { kinopoisk: 12345 }
  });
</script>

🔧 Параметры kinobox()

ПараметрТипОписание
selectorstringCSS-селектор контейнера плеера
search.kinopoisknumberID фильма/сериала на Кинопоиске
search.imdbstringIMDB ID (например, tt0111161)
search.titlestringНазвание для поиска
search.yearnumberГод выхода (для уточнения поиска)

2. Характер сервиса

KinoBox.tv является агрегатором — технической оболочкой, которая встраивает видеоплееры сторонних платформ через публичные API и iframe. Сервис не хранит, не загружает и не распространяет видеофайлы напрямую. Весь видеоконтент размещён на серверах третьих сторон.

3. Возрастные ограничения

Сервис предназначен для лиц старше 18 лет. Используя KinoBox.tv, вы подтверждаете достижение совершеннолетия по законодательству вашей страны.

4. Запрещённые действия

При использовании сервиса запрещается:

  • Использовать автоматизированные системы (боты, парсеры) для сбора данных
  • Предпринимать попытки взлома или нарушения работы сервиса
  • Распространять ссылки на контент, нарушающий законодательство
  • Использовать сервис в коммерческих целях без разрешения

5. Ограничение ответственности

KinoBox.tv не несёт ответственности за содержимое видеоматериалов, предоставляемых третьими сторонами. Сервис предоставляется «как есть» без каких-либо гарантий бесперебойной работы.

Политика конфиденциальности

Последнее обновление: 1 января 2025 года

1. Что мы собираем

KinoBox.tv автоматически собирает ограниченный набор технических данных:

  • IP-адрес (для геолокации источников)
  • Тип браузера и операционной системы
  • Страницы, которые вы посещаете
  • Время и дата визита

Мы не собираем персональные данные (имя, e-mail, номер телефона) без вашего явного согласия.

2. Cookies

Сайт использует технические cookies для корректной работы (например, сохранение выбранного источника плеера). Сторонние сервисы аналитики могут устанавливать собственные cookies.

3. Аналитика

Для улучшения сервиса используется Яндекс.Метрика. Данная служба собирает анонимную статистику посещений согласно политике конфиденциальности Яндекса. Вы можете отключить сбор данных, установив расширение для блокировки трекеров.

4. Сторонние плееры

Встраиваемые плееры (Collaps, Kodik, Alloha и другие) являются независимыми сервисами и имеют собственные политики конфиденциальности. KinoBox.tv не контролирует данные, которые они собирают.

5. Передача данных третьим лицам

Мы не продаём, не сдаём в аренду и не передаём ваши данные третьим лицам, за исключением случаев, предусмотренных законодательством.

6. Ваши права

Вы вправе запросить информацию о ваших данных или потребовать их удаления, написав на privacy@kinobox.tv.

7. Изменения политики

Мы можем обновлять эту политику. О существенных изменениях уведомим на главной странице.
